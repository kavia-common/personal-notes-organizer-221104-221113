# Notes Manager Frontend

- Configure backend URL using REACT_APP_BACKEND_URL or REACT_APP_API_BASE (defaults to http://localhost:3001).
- Start backend on port 3001 and frontend on port 3000.
- UI includes:
  - Sticky header
  - Notes grid
  - Floating + button for new note
  - Modal for create/edit
- Optimistic updates with error rollback on delete failures.
